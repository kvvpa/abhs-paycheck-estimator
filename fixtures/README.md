# Fixtures

This folder contains anonymized example data used for testing and demo.

## What is included
- `fixtures/stubs/*.json` are extracted numbers from anonymized ADP pay stubs.
- They are used to verify the calibration math (net profile) without storing personal identifiers.

## What is NOT included
- No names, account numbers, or identifiers.
- No full PDFs are required for the app to work.

If you want to add your own examples, follow the schema in the JSON files.
